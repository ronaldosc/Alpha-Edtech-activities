Existem atributos que modificam os controles exibidos ao usuário, em especial o atributo controlslist é funcional apenas no navegador Chrome por ser uma API otimizada especificamente para isso, conforme documentação detalhada.
Assim há como bloquear em separado: tela cheia, velocidade de reprodução, download. Um bom exemplo de uso deses recursos se encontra nessa página.
Nos demais navegadores se implementa alguns desses bloqueios de recursos do controle por funções da linguagem JavaScript.
